<static> createProgramList(dooh, intervalOfSelectingUGC, intervalOfDoohProgrames, created_cb)

Automatically selects applied UGC items (based on a predefined rule) and put them in "to-be-played" list for a specific DOOH.

This method will first ask ScalaMgr about the available time intervals in which Miix system can play its UGC content. It will then generate time slots base on a specific rule, and then fill them by pickeing up UGC items from a sorted list generated from censerMgr.
Parameters:
Name Type Description
dooh Number The ID of the DOOH upon which the selected UGCs are played
intervalOfSelectingUGC Object An object specifying the starting and ending of of the time interval for scheduleMgr to select the applied UGC items
  • start: Date()-readable string specifying the start of the interval
  • end: Date()-readable string specifying the end of the interval
For example, {start: '2013/6/21 8:30', end: '2013/6/21 13:00'}
intervalOfDoohProgrames Object An object specifying the starting and ending of of the time interval which the generated schedule covers
  • start: Date()-readable string specifying the start of the interval
  • end: Date()-readable string specifying the end of the interval
For example, {start: '2013/6/22 8:30', end: '2013/6/22 13:00'}
created_cb Function The callback function called when the result program list is created.
The function signature is created_cb(resultProgramList, err) :
  • resultProgramList: An array of objects containing program info:
    • id: A number specifying the ID of a program time slot item
    • interval: An object specifying the starting and ending time of the program item
      • start: Date()-readable string specifying the start of the interval
      • end: Date()-readable string specifying the end of the interval
    • ugc: A number specifying the ID of the UGC contained in this program. (This is actually the id of items store in ugc collection.)
    For example,
    [{_id:43524, interval:{start:'2013/6/22 8:30:00', end:'2013/6/22 8:30:30'}, ugc:48593},
    {_id:43525, interval:{start:'2013/6/22 8:30:00', end:'2013/6/22 8:30:30'}, ugc:48353},
    {_id:43544, interval:{start:'2013/6/22 8:30:00', end:'2013/6/22 8:30:30'}, ugc:43593}]
  • err: error message if any error happens

<static> updateProgramList(dooh, intervalOfDoohProgrames, updated_cb)

Update the programs (of a specific DOOH) of a specific interval.

This method will ask ScalaMgr about the latest available time intervals (in which Miix system can play its UGC content) and then re-schedule the programs accordingly.
Parameters:
Name Type Description
dooh Number The ID of the DOOH whose programs are to be updated
intervalOfDoohProgrames Object An object specifying the starting and ending of of the time interval in which the scheduled programs will be updated
  • start: Date()-readable string specifying the start of the interval
  • end: Date()-readable string specifying the end of the interval
For example, {start: '2013/6/22 8:30', end: '2013/6/22 13:00'}
updated_cb Function The callback function called when the program list is updated.
The function signature is updated_cb(listOfProgramsOutOfSchedule, err) :
  • listOfProgramsOutOfSchedule: An array of objects containing program info:
    • programTimeSlot: A number specifying the ID of the program item which is removed from the schedule after updating
    • ugc: A number specifying the ID of the UGC contained in this removed program. (This is actually the id of items store in ugc collection.)
    For example,
    [{_id:43524, ugc:48593},
    {_id:43525, ugc:48353},
    {_id:43544, ugc:43593}]
  • err: error message if any error happens
