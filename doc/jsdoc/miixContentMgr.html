<static> addMiixImage(imgBase64, ugcProjectID, ugcInfo, cbOfAddMiixImage)

Add a Miix image to system.

Miix image content body (in base64 format) is saved as a PNG and put onto AWS S3; the complete set of content info is save to UGC db
Parameters:
Name Type Description
imgBase64 String image date with base64 format
ugcProjectID String Project ID of this UGC
ugcInfo Object An object containing UGC info:
  • ownerId: ownerId An object containing owner's id info:
    • _id: owner's member ID (hex string representation of its ObjectID in MongoDB)
    • fbUserId: owner's Facebook user ID
  • contentGenre: it is normally the template (id) that this UGC uses
  • title: title of UGC
cbOfAddMiixImage Function Callback function called when adding operation is done

<static> preAddMiixMovie(ugcProjectID, ugcInfo, cbOfPreAddMiixMovie)

Add some info some preliminary info of Miix movie to UGC db and ask AE Server to render Miix video.

These info are for clinet side to show some dummy icon for Miix movie which is under rendering
Parameters:
Name Type Description
ugcProjectID String Project ID of this UGC
ugcInfo Object An object containing UGC info:
  • ownerId: ownerId An object containing owner's id info:
    • _id: owner's member ID (hex string representation of its ObjectID in MongoDB)
    • fbUserId: owner's Facebook user ID
  • contentGenre: it is normally the template (id) that this UGC uses
  • customizableObjects: an array of objects describing the customizable objects
  • title: title of UGC. This title will be used when it is posted on YouTube
cbOfPreAddMiixMovie Function Callback function called when adding operation is done
